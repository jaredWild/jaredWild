## Featured Projects

### Movement Complexity Analysis

**Master's Thesis – University of Tartu**

Investigated the relationship between human movement complexity and creativity through the analysis of virtual reality telemetry, natural language processing, and statistical modeling.
The project combined behavioral data science, spatial analytics, and cognitive computing to process data from 104 participants in a controlled VR environment. Raw 3D positional tracking data was transformed into 18 movement-complexity metrics, while creativity measures were evaluated using NLP-based semantic distance methods adapted for Estonian-language responses.

Key contributions:
•	Designed and implemented the end-to-end data processing pipeline
•	Engineered movement-complexity features from raw VR trajectory data
•	Applied NLP techniques to quantify creativity measures
•	Developed statistical validation frameworks to isolate confounding factors
•	Evaluated machine learning feasibility using PCA and clustering methods
The project ultimately demonstrated that exploratory behavior existed along a continuous spectrum rather than forming discrete clusters, supporting statistical analysis approaches over more complex machine learning models.
